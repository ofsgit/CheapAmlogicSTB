# P212 model
a typical p212 model is the Amlogic S905X-W, it has 1-2GB of ram and 6-8GB Storage with Ethernet, SPDIF, HDMI Out, AV Out. it also has 2 USB Port and 1 SD Card slot
for Connectivity, it tends to use the RealTek RTL8189FTV, which talks through Hi-Speed Bus. the RealTek RTL8189FTV supports WiFi-Direct and its WiFi 4 MIMO1x1
